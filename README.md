# kamoku-form-templates
「ドットインストール」で「Experss入門」を学習後、以前作った大学の単位判定アプリ「tani-checker」の科目表を動的に生成するテンプレートを作りました。 
Expressのアプリケーション・スケルトンから、publicフォルダとviewsフォルダ、routesフォルダを書き換えて作りました。 
仮想のデータベースからの授業データや科目データをもとに科目表の中身を動的に生成することを目標としました。 
科目表の階層構造を再現するテンプレート作りに苦労しました。 
テンプレートが複雑になってしまったので、もう少し簡単な方法で再現できることが今後の改善点だと思っています。 
今後は、MVCモデルについて理解するためにRuby on Railsの学習を進めたいと思います。
